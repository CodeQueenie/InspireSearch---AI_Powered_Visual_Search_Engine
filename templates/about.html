About InspireSearch

InspireSearch is an AI-powered visual search engine that helps you find visually similar images using deep learning technology.

Technology Stack

This project demonstrates several key technologies and concepts:

Deep Learning

Uses ResNet50, a pre-trained convolutional neural network, to extract high-dimensional feature vectors from images.

Similarity Search

Leverages Faiss, a library developed by Facebook AI Research for efficient similarity search and clustering of dense vectors.

Web Framework

Built with Flask, a lightweight WSGI web application framework in Python, making it easy to create web services.

Cloud Deployment

Designed to be easily deployed on AWS Free Tier services like Elastic Beanstalk or EC2.

How It Works

  1. When you upload an image, it's processed by a pre-trained ResNet50 model.
  2. The model extracts a 2048-dimensional feature vector that represents the visual characteristics of your image.
  3. This feature vector is compared against pre-computed vectors from our image dataset using Faiss.
  4. The system finds the most similar images based on the Euclidean distance between vectors.
  5. The top 5 most similar images are displayed, along with their similarity scores.
